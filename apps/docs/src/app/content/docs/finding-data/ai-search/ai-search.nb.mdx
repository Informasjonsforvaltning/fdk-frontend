# Kunstig intelligens-søk (KI-søk)

KI-søket bruker metadata fra Data.norge.no og hjelper deg med å finne datasett. Ved å bruke naturlig språk kan du søke fritt uten å måtte kjenne til spesifikke datasettnavn, fagtermer eller tekniske formater. Prøv f.eks. "Hvilke områder i Norge er vernet?"

Du kan lese teknisk informasjon om KI-søket på data.norge.no her.
