# Kunstig intelligens-søk (KI-søk)

KI-søket bruker metadata frå Data.norge.no og hjelper deg med å finne datasett. Ved å bruke naturleg språk kan du søkje fritt utan å måtte kjenne til spesifikke datasettnamn, fagomgrep eller tekniske format. Prøv til dømes "Kva område i Noreg er verna?"

Du kan lese teknisk informasjon om data.norge.no sitt KI-søk på [data.norge.no sine Github-sider](https://github.com/Informasjonsforvaltning/fdk-llm-search-service).
