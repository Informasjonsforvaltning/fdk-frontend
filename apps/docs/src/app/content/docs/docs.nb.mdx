# Kom i gang

Velkommen til veiledningen for data.norge.no! Her får du hjelp til å komme i gang med å finne og dele data. Utforsk våre guider for å lære hvordan du enkelt kan [finne data](/docs/finding-data) som passer dine behov, eller hvordan du kan [dele data](/docs/sharing-data) for å bidra til en mer åpen og datadrevet offentlig sektor.
