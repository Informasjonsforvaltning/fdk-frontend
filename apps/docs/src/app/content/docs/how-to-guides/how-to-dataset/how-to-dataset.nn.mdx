# Korleis lage og publisere ein datasettbeskriving – frå A til Å

Denne veiledninga tar deg gjennom alle stega som trengs for å nå målet, frå å finne ut kva datasett du vil beskrive, til å få det publisert til data.norge.no! Vi går gjennom:

-   1. Forberedelsar og kartlegging.
-   2. Korleis lage sjølve datasettbeskrivinga.
-   3. Korleis registrere ei datakjelde som data.norge.no kan hente beskrivinga frå.
-   4. Korleis starte ein høsting slik at data.norge.no hentar og publiserer beskrivinga di.

Undervegs viser vi konkrete døme på korleis du gjer dei ulike stega basert på datasettet "KI-prosjekt i offentleg sektor" frå Digdir.
