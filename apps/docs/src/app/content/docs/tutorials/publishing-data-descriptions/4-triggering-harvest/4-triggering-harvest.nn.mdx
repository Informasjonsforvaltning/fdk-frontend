# Korleis starte ei hausting

<Alert size='sm'>
    Du må først ha [lagd
    beskrivingar](/docs/tutorials/publishing-data-descriptions/2-creating-descriptions-registration) i Data.norge.no si
    registreringsløysing, eller [sett opp eit
    hausteendepunkt](/docs/tutorials/publishing-data-descriptions/3-publishing-descriptions) som Data.norge.no kan
    hauste frå.
</Alert>

## Steg 1: Logg inn

Logg inn i [administrasjonsgrensesnittet for hausting](https://admin.fellesdatakatalog.digdir.no/data-sources)

## Steg 2: Finn datakjelde

Finn den datakjelda du ønskjer å starte hausting av.

![Eksempel på kjelde](https://cms.fellesdatakatalog.digdir.no/uploads/data_source_entry_d25303217f.png)

## Steg 3: Start hausting

![Knapp som sier "Høst"](https://cms.fellesdatakatalog.digdir.no/uploads/harvesting_harvest_button_d1e18ff981.png)

Klikk på knappen «Høst».

![Melding som seier at høsteførespurnad vart sendt](https://cms.fellesdatakatalog.digdir.no/uploads/harvesting_harvest_request_sent_8f75b28d4a.png)

Ein bekreftelsesmelding vil visast. Det kan ta noko tid før endringane dine visast på data.norge.no.
