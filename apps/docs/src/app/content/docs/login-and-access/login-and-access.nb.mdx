# Hvordan få tilgang til data.norge.no sine løsninger

|     | Aktivitet                                                                                                                                                                                         | Ansvarleg |
| --- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| 1   | Gå til Altinn for å [lese om hvordan du får/gir tilgang til Felles datakatalog](https://info.altinn.no/skjemaoversikt/digitaliseringsdirektoratet/felles-datakatalog/)                            | Kunde     |
| 2   | Logg inn med ID-porten på [Felles datakatalog publisering](https://data.norge.no/publishing)                                                                                                      | Kunde     |
| 3   | Bruksvilkår må godtas første gang din virksomhet tar i bruk Felles datakatalog. Dette krever tilgangen virksomhetsadministrator. Les [bruksvilkår](https://data.norge.no/publishing/terms-of-use) | Kunde     |
| 4   | Nå er du klar til å registrere og/eller høste beskrivelser av virksomhetens data.                                                                                                                 | Kunde     |

## Datalandsbyen

[Datalandsbyen](https://datalandsbyen.norge.no/) er et nettforum hvor man kan etterspørre data, dele erfaringer og spørre om råd som gjelder datadeling og informasjonsforvaltning.

### Ta i bruk

For å ta i bruk Datalandsbyen må du registrere deg. Dette gjøres enkelt ved å følge stegene under.
NB. Alle kan lese alt innhold i Datalandsbyen uten å være registrert.

|     |                                                                                                                                                                     |
| --- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1   | Registrer bruker i [Felles brukerhåndtering](https://user.difi.no/auth/realms/difi/login-actions/authenticate?client_id=felles-datakatalog-prod&tab_id=VWHF5qpHL-g) |
| 2   | Godkjenn [samtykke og brukervilkår](https://datalandsbyen.norge.no/consent) for nettforumet                                                                         |
| 3   | Første innlegg som publiseres av en ny bruker i forumet må godkjennes av en administrator.                                                                          |
