# Kontakt oss

<Ingress size='xs'>
    Har du noen spørsmål, innspill eller trenger hjelp? Ta gjerne kontakt med oss på følgende måter:
</Ingress>

1. Send oss en e-post på [fellesdatakatalog@digdir.no](mailto:fellesdatakatalog@digdir.no)
2. Still spørsmål, etterspør data, og diskuter med oss og andre brukere på vårt åpne diskusjonsforum, [Datalandsbyen](https://datalandsbyen.norge.no/)
3. Følg oss, og se vår åpne kildekode på [Github](https://github.com/Informasjonsforvaltning)
