# How to start a harvest

<Alert
    size="sm"
>
You must first have [created descriptions](/docs/tutorials/publishing-data-descriptions/2-creating-descriptions-registration) in Data.norge.no's registration solution, or [set up a harvesting endpoint](/docs/tutorials/publishing-data-descriptions/3-publishing-descriptions) that Data.norge.no can harvest from.
</Alert>

## Step 1: Log in

Log in to the [admin interface for harvesting](https://admin.fellesdatakatalog.digdir.no/data-sources).

## Step 2: Find data source

Find the data source you want to start harvesting from.

![Example data source](https://cms.fellesdatakatalog.digdir.no/uploads/data_source_entry_d25303217f.png)

## Step 3: Start harvest

![Button that says "HÃ¸st"](https://cms.fellesdatakatalog.digdir.no/uploads/harvesting_harvest_button_d1e18ff981.png)

Click the "Harvest" button.

![Message that says that a harvest request was sent](https://cms.fellesdatakatalog.digdir.no/uploads/harvesting_harvest_request_sent_8f75b28d4a.png)

A confirmation message will appear. It may take some time before your changes are displayed on data.norge.no.
