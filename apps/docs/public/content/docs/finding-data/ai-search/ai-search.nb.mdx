# Kunstig intelligens-søk (KI-søk)

KI-søket bruker metadata fra Data.norge.no og hjelper deg med å finne datasett. Ved å bruke naturlig språk kan du søke fritt uten å måtte kjenne til spesifikke datasettnavn, fagtermer eller tekniske formater. Prøv f.eks. "Hvilke områder i Norge er vernet?"

Du kan lese teknisk informasjon om data.norge.no sitt KI-søk [data.norge.no sine Github-sider](https://github.com/Informasjonsforvaltning/fdk-llm-search-service)

<Alert>
  Tjenesten lagrer alle spørringer og antall treff i en database. Informasjonen brukes til å analysere søkemønstre og forbedre søketjenesten i fremtiden. Dataene kan ikke knyttes til enkeltpersoner, og det lagres ingen personopplysninger utover dette. Ettersom søkestrengene blir lagret, anbefaler vi at personopplysninger ikke oppgis i søkefeltet.
</Alert>
