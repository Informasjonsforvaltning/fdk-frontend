# Specifications, Resources, and Tools

## Specifications

Data.norge.no relies on several standards and specifications. Which standard/specification you should use depends on the type of resource (dataset, concept, service, etc.) you are describing.

-   Standard for describing datasets, data services og datacatalogs ([DCAT-AP-NO](https://data.norge.no/specification/dcat-ap-no))
-   Governance standard for concept descriptions ([SKOS-AP-NO-Begrep](https://data.norge.no/specification/skos-ap-no-begrep))
-   Specification for describing information models ([ModelDCAT-AP-NO](https://data.norge.no/specification/modelldcat-ap-no))
-   Specification for describing services and events ([CPSV-AP-NO](https://data.norge.no/specification/cpsv-ap-no))

## Tools Offered by Data.norge.no

Data.norge.no provides several open libraries and tools that can assist in creating descriptions.

-   [datacatalogtordf](https://github.com/Informasjonsforvaltning/datacatalogtordf): Python library that maps data catalogs to an RDF graph according to DCAT-AP-NO.
-   [concepttordf](https://github.com/Informasjonsforvaltning/concepttordf): Python library that maps concept catalogs to an RDF graph according to SKOS-AP-NO.
-   [modelldcatnotordf](https://github.com/Informasjonsforvaltning/modelldcatnotordf): Python library that maps information model catalogs to RDF graph according to ModelDCAT-AP-NO.
-   [servicecatalogtordf](https://github.com/Informasjonsforvaltning/servicecatalogtordf): Python library that maps service and/or event catalogs to an RDF graph according to CPSV-AP-NO.
-   [oastodcat](https://github.com/Informasjonsforvaltning/oastodcat): Python library that transforms an OpenAPI specification into instances of dcat:DataService (DCAT-AP-NO).
-   [atlasdcat](https://github.com/Informasjonsforvaltning/atlasdcat): Python library that maps from Apache Atlas Glossary terms to DCAT metadata and vice versa.
-   [jsonschematordf](https://github.com/Informasjonsforvaltning/jsonschematordf): Python library that maps from JSON Schema to an RDF graph according to ModelDCAT-AP-NO.
-   [Validation tool](https://data.norge.no/validator): Validates dataset descriptions (DCAT-AP-NO), concept descriptions (SKOS-AP-NO), and service and event descriptions (CPSV-AP-NO) in RDF format.
