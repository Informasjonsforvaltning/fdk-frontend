# Korleis få tilgang til data.norge.no sine løysingar

|     | Aktivitet                                                                                                                                                                                          | Ansvarleg |
| --- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| 1   | Gå til Altinn for å [lese om korleis du får/gir tilgang til Felles datakatalog](https://info.altinn.no/skjemaoversikt/digitaliseringsdirektoratet/felles-datakatalog/)                             | Kunde     |
| 2   | Logg inn med ID-porten på [Felles datakatalog publisering](https://data.norge.no/publishing)                                                                                                       | Kunde     |
| 3   | Bruksvilkår må godkjennast første gong verksemda di tar i bruk Felles datakatalog. Dette krev tilgang som verksemdsadministrator. Les [bruksvilkår](https://data.norge.no/publishing/terms-of-use) | Kunde     |
| 4   | No er du klar til å registrere og/eller hauste beskrivingar av verksemda sine data.                                                                                                                | Kunde     |

## Datalandsbyen

[Datalandsbyen](https://datalandsbyen.norge.no/) er eit nettforum der ein kan etterspørje data, dele erfaringar og spørje om råd som gjeld datadeling og informasjonsforvaltning.

### Ta i bruk

For å ta i bruk Datalandsbyen må du registrere deg. Dette gjer du enkelt ved å følgje stega nedanfor.
NB. Alle kan lese alt innhald i Datalandsbyen utan å vere registrert.

|     |                                                                                                                                                                     |
| --- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1   | Registrer brukar i [Felles brukarhandtering](https://user.difi.no/auth/realms/difi/login-actions/authenticate?client_id=felles-datakatalog-prod&tab_id=VWHF5qpHL-g) |
| 2   | Godkjenn [samtykke og bruksvilkår](https://datalandsbyen.norge.no/consent) for nettforumet                                                                          |
| 3   | Første innlegg som blir publisert av ein ny brukar i forumet må godkjennast av ein administrator.                                                                   |
