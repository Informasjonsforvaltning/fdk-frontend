# Kontakt oss

<Ingress size='xs'>Har du spørsmål, innspel eller treng hjelp? Ta gjerne kontakt med oss på følgjande måtar:</Ingress>

1. Send oss ein e-post på [fellesdatakatalog@digdir.no](mailto:fellesdatakatalog@digdir.no)
2. Still spørsmål, etterspør data, og diskuter med oss og andre brukarar på vårt opne diskusjonsforum, [Datalandsbyen](https://datalandsbyen.norge.no/)
3. Følg oss, og sjå vår opne kjeldekode på [GitHub](https://github.com/Informasjonsforvaltning)

<Alert size='md'>
    <strong>Viktig:</strong> Spørsmål om spesifikke datasett, API-ar eller annan data som verksemder har publisert på
    denne portalen, må takast direkte med den ansvarlege verksemda.
</Alert>
