# Tjenester og hendelser

<Ingress>
    Formålet med Tjeneste- og hendelseskatalogen er å gi en oversikt over tjenester, og hendelser som kan utløse eller
    utløses av disse. Tjenestene som beskrives i katalogen er tjenester som tilbys av eller på vegne av offentlig sektor,
    samt tjenester som leveres av andre og som er viktige for å kunne realisere sammenhengende tjenester.
</Ingress>
