# Tenester og hendingar

<Ingress>
    Formålet med Teneste- og hendingskatalogen er å gi ei oversikt over tenester, og hendingar som kan utløysa eller
    utløysast av desse. Tenestene som blir beskrivne i katalogen er tenester som blir tilbodne av eller på vegner av
    offentleg sektor, og dessutan tenester som blir leverte av andre og er spesielt viktige for å kunne realisera
    samanhengande tenester.
</Ingress>
