# API-katalogen

<Ingress>
    API-katalogen i Felles datakatalog sikrar enkel tilgang til distribuerte data. Gjennom API-katalogen kan ein hente ut maskinlesbare opplysningar om alle adressene (URL-ar) som iverkset eit spesifikt API. API-katalogen inneheld ei oversikt over skildringar av grensesnitta til data og standardar som verksemder har gjort tilgjengelege. Tilgjengelege API-ar er nødvendige for å oppnå effektiv utnytting av data og gir betre kvalitet for tenester.
</Ingress>

## Kva informasjon finn eg i API-katalogen?

Alle API-ane som er publiserte i Felles datakatalog følgjer [Standard for datatenester](https://data.norge.no/specification/dcat-ap-no/#Datatjeneste).

Nedanfor kan du sjå kva du kan få informasjon om:

-   Skildring av API-et
-   Format du kan få API-et i
-   Skildring av korleis du kan ta i bruk datatenesta
-   Tilknytte datasett-skildringar
-   Kontaktinformasjon til eigar av API

## Relasjon til andre katalogar

**Datasettkatalogen**  
Eit API kan ha relasjon til eitt eller fleire datasett. Det betyr at datasett(a) kan nåast gjennom API-et dersom datasett er oppgitt.

**Informasjonsmodellkatalogen**  
Informasjonsmodellen kan vere oppgitt i API-skildringa. Det betyr at du kan sjå korleis datatenesta er strukturert.
