# Hvordan lage og publisere en datasettbeskrivelse – fra A til Å

Denne veiledningen tar deg gjennom alle trinnene som trengs for å komme i mål, fra å finne ut hvilket datasett du vil beskrive, til å få det publisert til data.norge.no! Vi går igjennom:

- Forberedelser og kartlegging.
- Hvordan lage selve datasettbeskrivelsen.
- Hvordan registrere en datakilde som data.norge.no kan hente beskrivelsen fra.
- Hvordan starte en høsting sånn at data.norge.no henter og publiserer beskrivelsen din.

Underveis viser vi konkrete eksempler på hvordan gjøre de ulike stegene basert på datasettet "KI-prosjekter i offentlig sektor" fra Digdir.
