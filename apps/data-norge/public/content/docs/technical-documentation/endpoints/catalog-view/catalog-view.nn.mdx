# Catalog view-endepunkt

<Ingress size='xs'>Oppslags-endepunkt mot eigen verksemd sin katalog i registreringsløysing. Gjennom dette endepunktet kan verksemder hente ut all informasjon om ressursar som verksemden har registrert i registreringsløysing. 
Førebels er det berre mogleg å nytte dette endepunktet mot omgrepskatalogen.</Ingress>

## Endepunkt i ulike miljø

* Produksjonsmiljø: https://catalog-view.api.fellesdatakatalog.digdir.no
* Demomiljø:  https://catalog-view.api.demo.fellesdatakatalog.digdir.no

## Dokumentasjon

[OpenAPI-spesifikasjon for catalog view-endepunktet (Github)](https://github.com/Informasjonsforvaltning/catalog-view-api/blob/main/openapi.yaml) 

## Slik får du tilgang

API-et er beskytta med Maskinporten. [Ta kontakt med oss](/contact) om du ønsker tilgang for å hente ut di verksemd sine beskrivingar frå registreringsløysinga.

Gå til [sida om Maskinporten på Samarbeidsportalen](https://samarbeid.digdir.no/maskinporten/maskinporten/25) for å lese meir om kva Maskinporten er og korleis du tar tenesten i bruk som konsument.
