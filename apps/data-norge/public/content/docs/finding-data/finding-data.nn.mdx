# Finne data

<Ingress size='xs'>
    Data.norge.no gir ei omfattande oversikt over datasett, omgrep, API-ar, informasjonsmodellar, tenester og hendingar
    i offentleg sektor, inkludert data med juridiske avgrensingar som teieplikt eller personopplysningar. Hensikta med
    denne oversikta er å gjere det lettare for deg å finne, forstå og ta i bruk data frå offentleg sektor. For å finne
    fram blant alle beskrivingane, tilbyr Data.norge.no ulike måtar å søkje på. Om du ikkje finn dataene du leitar
    etter, kan du registrere ein etterspurnad.
</Ingress>

## Søk etter data

Data.norge.no tilbyr tre forskjellige måtar å søkje etter data på: Standard tekstsøk med filtrering, kunstig intelligens-søk og SPARQL-spørjing.

### 1. Standard søk med filter

På data.norge.no/search-all finn du eit tekstsøk med moglegheiter for filtrering. Her kan du søkje etter titlar, beskrivingar og søkjeord i databeskivingane. For å avgrense søket ditt, kan du bruke ulike filter. Kva filter som blir tilbydd avheng av ressursstype. Du vel ressursstype ved å bruke fanene under søkjefeltet. Resultatet kan sorterast etter relevans eller publiseringsdato.

[Gå til standard søk med filter](/search-all)

### 2. Søk med kunstig intelligens

Med KI-søket kan du stille spørsmål i naturleg språk og få resultat frå data.norge.no basert på det. Dette er ideelt når du ikkje veit nøyaktig kva du leitar etter eller ønskjer å utforske data på ein meir intuitiv måte.

[Les meir om KI-søk](/docs/finding-data/ai-search) | [Prøv KI-søket](https://data.norge.no)

### 3. SPARQL-spørjing

SPARQL-tjenesten er det kraftigaste og mest fleksible verktøyet for å søkje i katalogen. Dette er eit verktøy der du kan utføre dine eigne spørjingar i SPARQL på RDF-grafane til data.norge.no.

[Les meir om SPARQL](/docs/finding-data/sparql) | [SPARQL-sandkasse](https://data.norge.no/nb/sparql)

<Alert size='sm'>
    Du kan lese meir om å finne og vurdere data på [Digitaliseringsdirektoratet sine
    sider.](https://www.digdir.no/datadeling/finne-og-vurdere-data/2256#hvor_kan_du_finne_dataene)
</Alert>

## Kven har ansvar for kva på data.norge.no

### Datatilbydarane

Verksemdene som eig dataene som er skildra på data.norge.no har ansvar for sjølve innhaldet i beskrivingane og for at dataene du får tilgang til stemmer overeins med beskrivingane.

Om du finn noko feil med beskrivingane eller tilgangen til dataene, kan du kontakte verksemda som eig beskrivinga direkte. Kontaktinformasjon vil som regel vere knytt til beskrivinga. Om du er logga inn i Datalandsbyen, kan du òg legge igjen ein kommentar på sjølve beskrivinga på data.norge.no. Denne kommentaren vil visast både på data.norge.no og i Datalandsbyen. Slike kommentarar og påfølgjande diskusjon kan vere nyttig for andre brukarar som treng dei same dataene som deg.

### Digitaliseringsdirektoratet

Digitaliseringsdirektoratet har ansvar for informasjon om når beskrivingar vart publiserte på data.norge.no, metadatakvalitet, Datajegeren, KI-oversikt, informasjonssider og at dei tekniske løysingane på data.norge.no og Datalandsbyen fungerer som dei skal. Digitaliseringsdirektoratet følgjer òg opp kommentarar i Datalandsbyen og purrar på verksemder som ikkje svarar på kommentarar knytt til beskrivingar i katalogen.
