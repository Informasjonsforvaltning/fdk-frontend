# Ressursteneste

Ressurstenesta gjev deg direkte tilgang til ressursar frå data.norge.no gjennom REST API. Du kan hente ut datasett, omgrep, datatenester, informasjonsmodellar, tenester og hendingar, både som JSON og RDF-grafer.

Tenesta støttar fleire ressurstypar og gjev deg moglegheit til å hente ut både strukturerte data og RDF-representasjonar av ressursane.

[Les meir om ressursteneste-endepunktet](/nn/technical/api/resource-service)
