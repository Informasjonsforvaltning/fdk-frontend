# Ressurstjeneste

Ressurstjenesten gir deg direkte tilgang til ressurser fra data.norge.no gjennom REST API. Du kan hente ut datasett, begreper, datatjenester, informasjonsmodeller, tjenester og hendelser, både som JSON og RDF-grafer.

Tjenesten støtter flere ressurstyper og gir deg mulighet til å hente ut både strukturerte data og RDF-representasjoner av ressursene.

[Les mer om ressurtjeneste-endepunktet](/nb/technical/api/resource-service)
