# Tilgang til data

<Ingress size='xs'>Data kan gjøres tilgjengelig med ulike tilgangsnivåer.</Ingress>

## Trafikklyssystemet

Trafikklyssystemet samsvarer med [EUs klassifikasjon for tilgangsstyring](https://op.europa.eu/en/web/eu-vocabularies/concept-scheme/-/resource?uri=http://publications.europa.eu/resource/authority/access-right) og er enkelt å bruke.

| Tilgangsnivå                                   | Beskrivelse                                                                                                                                                                                                                                                                                                                                                                               |
| ---------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <Tag color="success">Allmenn tilgang</Tag>     | Tilgang tilgjengelig for alle. Tillatte begrensninger inkluderer brukerregistrering og API-nøkkelkrav, så lenge alle kan få tilgang på like vilkår. <br /><br/> **Eksempler:** Åpne data. Data med allmenn tilgang, men med bruksrestriksjoner, som kun ikke-kommersiell bruk, deling på samme vilkår eller forbud mot bearbeiding.                                                       |
| <Tag color="warning">Begrenset tilgang</Tag>   | Tilgang kun tilgjengelig under visse betingelser fastsatt av eier eller tilbyder. Betingelser kan inkludere betaling eller fortrolighetsavtale. Kategorien omfatter også data som ennå ikke er klassifisert som allmenn eller ikke-allmenn tilgang. <br /><br/> **Eksempler:** Forretningshemmeligheter, opphavsrettsbeskyttede data, data med betalingskrav eller eksklusivt medlemskap. |
| <Tag color="danger">Ikke-allmenn tilgang</Tag> | Tilgang begrenset av lovregulerte hensyn knyttet til sikkerhet, personvern eller lignende. Omfatter ressurser med sensitive opplysninger eller persondata. <br /><br/> **Eksempler:** Alminnelige og sensitive personopplysninger, taushetsbelagte data, informasjon av betydning for rikets sikkerhet.                                                                                   |

## Åpne data

Åpne data er datasett med allmenn tilgang og minst én distribusjon under en godkjent åpen lisens.

## Nasjonal Søknad- og interessemeldingssløsning

[Nasjonal Søknad- og interessemeldingssløsning](https://frontend-soknad.paas2.uninett.no/) er en felles plattform for å søke om tilgang til begrensede data eller melde interesse for upubliserte data. Målet er å forenkle og standardisere prosessen for både datatilbydere og brukere, slik at verdifulle data kan deles på en trygg og kontrollert måte.

Som en del av pilotprosjektet har Kudaf og Samferdselsdata utviklet en løsning som nå testes i praksis. Brukere kan sende inn forespørsler direkte via data.norge.no eller virksomheters datakataloger. Løsningen gjør det også mulig å registrere interesse for data som ennå ikke er klare for deling. Dette gir datatilbydere innsikt i etterspørselen og et bedre grunnlag for å vurdere om de skal prioritere tilgjengeliggjøring av disse dataene.
