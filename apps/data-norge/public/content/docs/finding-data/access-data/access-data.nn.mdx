# Tilgang til data

<Ingress size='xs'>Data kan gjerast tilgjengeleg med ulike tilgangsnivå.</Ingress>

## Trafikklyssystemet

Trafikklyssystemet samsvarer med [EUs klassifikasjon for tilgangsstyring](https://op.europa.eu/en/web/eu-vocabularies/concept-scheme/-/resource?uri=http://publications.europa.eu/resource/authority/access-right) og er enkelt å bruke.

| Tilgangsnivå                                    | Skildring                                                                                                                                                                                                                                                                                                                                                                    |
| ----------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <Tag color="success">Allmenn tilgang</Tag>      | Tilgang tilgjengeleg for alle. Tillatne avgrensingar inkluderer brukarregistrering og API-nøkkelkrav, så lenge alle kan få tilgang på like vilkår. <br /><br/> **Døme:** Opne data. Data med allmenn tilgang, men med bruksavgrensingar, som berre ikkje-kommersiell bruk, deling på same vilkår eller forbod mot endring.                                                   |
| <Tag color="warning">Avgrensa tilgang</Tag>     | Tilgang som berre er tilgjengeleg under visse vilkår fastsette av eigar eller tilbydar. Vilkåra kan inkludere betaling eller teiepliktsavtale. Kategorien omfattar òg data som enno ikkje er klassifiserte som allmenn eller ikkje-allmenn tilgang. <br /><br/> **Døme:** Forretningsløyndommar, opphavsrettsverna data, data med betalingskrav eller eksklusivt medlemskap. |
| <Tag color="danger">Ikkje-allmenn tilgang</Tag> | Tilgang avgrensa av lovpålagde omsyn knytt til tryggleik, personvern eller liknande. Omfattar ressursar med sensitive opplysningar eller persondata. <br /><br/> **Døme:** Alminnelege og sensitive personopplysningar, teiepliktige data, informasjon av betyding for rikstryggleiken.                                                                                      |

## Opne data

Opne data er datasett med allmenn tilgang og minst éi distribuering under ein godkjend open lisens.

## Nasjonal søknadsløysing

[Nasjonal søknadsløysing](https://frontend-soknad.paas2.uninett.no/) er ei felles plattform for å søkje om tilgang til avgrensa eller upubliserte data. Målet er å forenkle og standardisere prosessen for både datatilbydarar og brukarar, slik at verdifulle data kan delast på ein trygg og kontrollert måte.

Som ein del av pilotprosjektet har Kudaf og Samferdselsdata utvikla ei søknadsløysing som vert testa i praksis. Løysinga gjer det mogleg for brukarar å sende inn førespurnader direkte via data.norge.no eller organisasjonar sine datakatalogar. Datatilbydarar får samstundes betre oversikt over søknader og kan vurdere etterspurnaden etter data som enno ikkje er gjort tilgjengeleg.
