# SPARQL

SPARQL-tjenesten er det kraftigste og mest fleksible verktøyet for å søke i katalogen. Dette er et verktøy hvor du kan utføre egne spørringer i SPARQL på RDF-grafene til Data.norge.no. Her kan du spørre etter datasett, begreper, API-er og informasjonsmodeller.

## Hva er SPARQL?

SPARQL er et spørrespråk som brukes til å hente data lagret i RDF-format. RDF (Resource Description Framework) er en standardmodell for å beskrive data og ressurser på semantisk web. [Du kan lese mer om RDF her](/docs/sharing-data/rdf).

Du kan lese mer om SPARQL-spørrespråket i [W3C sin SPARQL-dokumentasjon](https://www.w3.org/TR/sparql11-overview/).

## SPARQL-verktøy

Du kan teste og utføre SPARQL-spørringer i vår interaktive SPARQL-sandkasse. Sandkassen gir deg et brukervennlig grensesnitt for å utforske norske offentlige data og finjustere dine spørringer.
