# SPARQL

SPARQL-tenesta er det kraftigaste og mest fleksible verktøyet for å søkje i katalogen. Dette er eit verktøy der du kan utføre eigne spørjingar i SPARQL på RDF-grafane til data.norge.no. Her kan du spørje etter datasett, omgrep, API-ar og informasjonsmodellar.

## Kva er SPARQL?

SPARQL er eit spørjespråk som vert nytta til å hente data lagra i RDF-format. RDF (Resource Description Framework) er ein standardmodell for å skildre data og ressursar på semantisk web. [Du kan lese meir om RDF her](/docs/sharing-data/rdf).

Du kan lese meir om SPARQL-spørjespråket i [W3C sin SPARQL-dokumentasjon](https://www.w3.org/TR/sparql11-overview/).

## SPARQL-verktøy

Du kan teste og utføre SPARQL-spørjingar i vår interaktive SPARQL-sandkasse. Sandkassen gjev deg eit brukarvensleg grensesnitt for å utforske norske offentlege data og finjustere dine spørjingar.
