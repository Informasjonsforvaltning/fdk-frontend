# Kunstig intelligens-søk (KI-søk)

KI-søket brukar metadata frå Data.norge.no og hjelper deg med å finne datasett. Ved å bruka naturleg språk kan du søkje fritt utan å måtte kjenne til spesifikke datasettnamn, fagomgrep eller tekniske format. Prøv til dømes "Kva område i Noreg er verna?"

[Les meir om KI-søkjeendepunktet](/nn/technical/api/ai-search) | [Prøv KI-søket på Data.norge.no](https://data.norge.no/nn)

<Alert>
    Tenesta lagrar alle spurnader og tal treff i ein database. Informasjonen blir brukt til å analysere søkjemønster og
    forbetre søkjetenesta i framtida. Dataa kan ikkje knytast til enkeltpersonar, og det blir ikkje lagra
    personopplysningar utover dette. Ettersom søkjestrengane blir lagra, rår vi til at personopplysningar ikkje blir
    oppgitt i søkjefeltet.
</Alert>
