# Standard søk med filter

På data.norge.no/search-all finner du et tekstsøk med muligheter for filtrering. Dette er den mest brukervennlige søkemåten for å finne data på Data.norge.no.

## Hvordan søke

Her kan du søke etter titler, beskrivelser og søkeord i databeskrivelsene. Søket vil automatisk søke i alle tre feltene samtidig, og resultatene vil bli vektet slik at treff i tittel prioriteres høyest.

## Filtrering

For å avgrense søket ditt, kan du bruke ulike filtre. Hvilke filtre som tilbys avhenger av ressurstype. Du velger ressurstype ved å bruke fanene under søkefeltet. Eksempler på filtre inkluderer:

- **Datatema**: Filtrer på hvilket tema dataene tilhører (f.eks. miljø, helse, transport)
- **Tilgangsrettigheter**: Filtrer på om dataene er offentlige eller begrensede
- **Format**: Filtrer på datatype (f.eks. JSON, CSV, XML)
- **Åpne data**: Filtrer på om dataene er klassifisert som åpne data
- **Organisasjon**: Filtrer på hvilken organisasjon som eier dataene

## Sortering

Resultatet kan sorteres etter relevans eller publiseringsdato. Som standard sorteres resultatene etter relevans, hvor treff som matcher søket ditt best vises først.
