# Standard søk med filter

På data.norge.no/search-all finn du eit tekstsøk med moglegheiter for filtrering. Dette er den mest brukarvenlege søkjemåten for å finne data på Data.norge.no.

## Korleis søkje

Her kan du søkje etter titlar, beskrivingar og søkjeord i databeskivingane. Søket vil automatisk søkje i alle tre felt samtidig, og resultata vil bli vekta slik at treff i tittel prioriterast høgast.

## Filtrering

For å avgrense søket ditt, kan du bruke ulike filter. Kva filter som blir tilbydd avheng av ressursstype. Du vel ressursstype ved å bruke fanene under søkjefeltet. Døme på filter inkluderer:

- **Datatema**: Filtrer på kva tema dataa tilhøyrer (t.d. miljø, helse, transport)
- **Tilgangsrettar**: Filtrer på om dataa er offentlege eller avgrensa
- **Format**: Filtrer på datatype (t.d. JSON, CSV, XML)
- **Opne data**: Filtrer på om dataa er klassifisert som opne data
- **Organisasjon**: Filtrer på kva organisasjon som eig dataa

## Sortering

Resultatet kan sorterast etter relevans eller publiseringsdato. Som standard sorterast resultata etter relevans, der treff som passar søket ditt best vert viste først.
