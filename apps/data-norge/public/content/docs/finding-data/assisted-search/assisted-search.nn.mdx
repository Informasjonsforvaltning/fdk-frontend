# Hjelp til å finne data

<Ingress>
    Veit du ikkje om data du ser etter finst eller er det data du treng som ikkje er delte enno? Dersom datasettet ikkje er
    skildra på Data.norge.no eller på dataeigarane sine nettsider, bør du gå i dialog med dei som tilbyr dei og oppfordre dei
    til å dele. Du kan bruka Digdir sine tenester til dette, ved å spørje Datajegeren eller etterspørje datasett og API-ar
    i Datalandsbyen.
</Ingress>

## Datajegeren: Finn og få tilgang til offentlege datasett

Datajegeren er ei teneste som hjelper deg med å finne og få tilgang til datasett frå offentleg sektor. Sjølv om mange datasett er tilgjengelege via data.norge.no, er ikkje katalogen komplett.

Du kan be Datajegeren om hjelp via eit enkelt kontaktskjema. Her kan du skildre kva du leitar etter. Nokre felt er obligatoriske og nokre kan du velje å bruka for å spesifisere førespurnaden din. Når skjemaet vert sendt inn, går det direkte til tilsette i Digitaliseringsdirektoratet som undersøkjer kva etat som eig dataene og tek seg av den naudsynte kontakten for å skaffe dei.

[Gå til Datajegeren](/data-hunter)

## Datalandsbyen: Etterspørje datasett og API-ar

Datalandsbyen er eit nettforum dedikert til dialog mellom tilbydarar og konsumentar av data. Her kan du etterlyse datasett og API-ar du ynskjer tilgang til eller har spurnader om. Du kan òg be om utvidingar av eksisterande datasett og API-ar, til dømes meir tilgjengeleg data eller endringar i API-funksjonalitet.

[Gå til Datalandsbyen](https://datalandsbyen.norge.no)
