# Spesifikasjonar, ressursar og verktøy

## Spesifikasjonar

Data.norge.no baserer seg på flere standarder og spesifikasjoner. Kva standard/spesifikasjon du skal bruke avhenger av typen ressurs (datasett, omgrep, teneste, osv.) du skal beskrive.

- [DCAT-AP-NO](https://data.norge.no/specification/dcat-ap-no)  
  Standard for beskriving av datasett, datatenester og datakatalogar.
- [SKOS-AP-NO-Begrep](https://data.norge.no/specification/skos-ap-no-begrep)  
  Forvaltningsstandard for beskriving av omgrep.
- [ModellDCAT-AP-NO](https://data.norge.no/specification/modelldcat-ap-no)  
  Spesifikasjon for beskriving av informasjonsmodellar.
- [CPSV-AP-NO](https://data.norge.no/specification/cpsv-ap-no)  
  Spesifikasjon for beskriving av tenester og hendingar.

## Verktøy Data.norge.no tilbyr

Data.norge.no har fleire opne bibliotek og verktøy som kan hjelpe i arbeidet med å lage beskrivingar.

- [datacatalogtordf](https://github.com/Informasjonsforvaltning/datacatalogtordf)  
  Python-bibliotek som mapper datakatalogar til ein RDF-graf i samsvar med DCAT-AP-NO.
- [concepttordf](https://github.com/Informasjonsforvaltning/concepttordf)  
  Python-bibliotek som mapper omgrepskatalogar til ein RDF-graf i samsvar med SKOS-AP-NO.
- [modelldcatnotordf](https://github.com/Informasjonsforvaltning/modelldcatnotordf)  
  Python-bibliotek som mapper informasjonsmodellkatalogar til RDF-graf i samsvar med ModellDCAT-AP-NO.
- [servicecatalogtordf](https://github.com/Informasjonsforvaltning/servicecatalogtordf)  
  Python-bibliotek som mapper teneste- og/eller hendingeskatalogar til ein RDF-graf i samsvar med CPSV-AP-NO.
- [oastodcat](https://github.com/Informasjonsforvaltning/oastodcat)  
  Python-bibliotek som transformerer ein OpenAPI-spesifikasjon til instans(ar) av `dcat:DataService` (DCAT-AP-NO).
- [atlasdcat](https://github.com/Informasjonsforvaltning/atlasdcat)  
  Python-bibliotek som mapper Apache Atlas Glossary terms til DCAT metadata og omvendt.
- [jsonschematordf](https://github.com/Informasjonsforvaltning/jsonschematordf)  
  Python-bibliotek som mapper frå JSON Schema til ein RDF-graf i samsvar med ModellDCAT-AP-NO.
- [Valideringsverktøy](https://data.norge.no/validator)  
  Validerer datasettbeskrivingar (DCAT-AP-NO), omgrepsbeskrivingar (SKOS-AP-NO) og teneste- og hendingesbeskrivingar (CPSV-AP-NO) i RDF-format.
- [SPARQL-brukargrensesnitt](https://data.norge.no/sparql)  
  Verktøy som lar deg gjere SPARQL-spørjingar på innhaldet i Data.norge.no.

Om du oppdagar feil eller manglar i desse, opprett gjerne eit issue i repoet eller [kontakt oss](/contact).
