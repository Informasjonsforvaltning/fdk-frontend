# Nasjonalt tilgangspunkt for veg- og transportdata

![Transportportal.no](https://cms.fellesdatakatalog.digdir.no/uploads/transportportal_no_transportportal_no_2ca14c55c4.jpg)

Transportportal.no er det nasjonale tilgangspunktet for data om veg og transport i Noreg. Portalen er oppretta for å ha tilgang til alle veg-, transport- og mobilitetsdata frå same stad.

Portalen inneheld data om mellom anna veg, vegstatus, hendingar, reisetider, kollektivruter, rutetider, informasjon, varsling og mobilitetstenester som bysyklar. Desse dataa kan brukast til å lage eigne tenester. Ein kan òg kombinere data frå ulike datasett til å lage nye, framtidsretta, gode mobilitetstenester retta mot alle trafikantar.

I portalen kan du søkje opp datasett både på norsk og engelsk og få nok informasjon om datasettet til å forstå kva det dekkjer. Datasetta skal vere tilgjengelege på standardiserte og spesifiserte dataformat i tråd med ITS-direktivet og forordningane.

Portalen inneheld berre metadata om datasettet. Datakjeldene er plasserte hos kvar enkelt dataeigar, og metadataa inneheld lenkjer til desse. Ved å følgje lenkja til eit datasett vil ein finne vidare informasjon om datasettet og ulike aspekt knytte til nedlasting, bruk av dataa, eventuelle restriksjonar med omsyn til bruk/vidarebruk, betaling og liknande.

Datasett som er skildra i portalen skal helst vere gratis å bruke, men det er opna for at dataeigar kan ta eit vederlag for nokre typar data. Både offentlege og private verksemder er venta å bidra med datasett til portalen, og dette vil kunne vere verksemder som er vegeigarar, opererer transport- og mobilitetstenester så vel som formidlarar av trafikkinformasjon.
I datakatalogen til portalen finn du skildringar av og lenkjer til datasett og tenester om vegstatus, hendingar, reisetider, kollektivtrafikk, rutetider og mykje meir.

Alle datasetta er opne - bruk dei gjerne for å utvikle eigne tenester.

Alle i Noreg som har datasett eller tilbyr tenester knytte til vegtrafikk, kollektivtrafikk og mobilitetstilbod kan og bør leggje informasjon om desse ope tilgjengeleg i denne portalen, sjå nærare skildring om innlegging av data.

Alle land i Europa skal ha eit slikt tilgangspunkt for å få til samanhengande informasjonstenester for transport - i tråd med ITS-direktivet til EU.

Portalen inneheld hovudsakleg datasett og tenester som blir omfatta av forordningane til ITS-direktivet, til dømes:

- Trafikkinformasjon om hendingar, reisetider, vêr- og føreforhold
- Statiske data om vegnettet
- Rute- og sanntidsinformasjon om kollektivtrafikk
- Stoppestadregister for kollektivtrafikk
- Data om trafikken på vegnettet
- Informasjon om parkeringsplassar
- Data om «nye mobilitetsformer» som bysyklar, sparkesyklar, etc.
