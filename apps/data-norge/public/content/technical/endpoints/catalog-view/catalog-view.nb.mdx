# Catalog view-endepunkt

<Ingress size='xs'>Oppslags-endepunkt mot egen virksomhets katalog i registreringsløsning. Gjennom dette endepunktet kan virksomheter hente ut all informasjon om ressurser som virksomheten har registrert i registreringsløsning. 
Foreløpig er det bare mulig å bruke dette endepunktet mot begrepskatalogen.</Ingress>

## Endepunkt i ulike miljøer

* Produksjonsmiljø: https://catalog-view.api.fellesdatakatalog.digdir.no
* Demomiljø:  https://catalog-view.api.demo.fellesdatakatalog.digdir.no

## Dokumentasjon

[OpenAPI-spesifikasjon for catalog view-endepunktet (Github)](https://github.com/Informasjonsforvaltning/catalog-view-api/blob/main/openapi.yaml) 

## Slik får du tilgang

API-et er beskyttet med Maskinporten. [Ta kontakt med oss](/contact) hvis du ønsker tilgang for å hente ut din virksomhets beskrivelser fra registreringsløsningen.

Gå til [siden om Maskinporten på Samarbeidsportalen](https://samarbeid.digdir.no/maskinporten/maskinporten/25) for å lese mer om hva Maskinporten er og hvordan du tar løsningen i bruk som konsument.
