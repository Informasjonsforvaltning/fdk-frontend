# SPARQL API

<Ingress>
    The SPARQL endepoint is the most powerful search tool on data.norge.no, and allows you to search for all properties
    that exists in the catalogs, as long as you master the syntax.
</Ingress>

## Endpoints in different environments

- Production environment: https://data.norge.no/nb/sparql
- Demo environment: https://demo.fellesdatakatalog.digdir.no/sparql
