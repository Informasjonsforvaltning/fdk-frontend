# SPARQL API

<Ingress size='xs'>SPARQL-endepunktet er det kraftigste søkeverktøyet på data.norge.no og gir deg mulighet til å søke på alle egenskaper som finnes i katalogene, så lenge du behersker syntaksen.</Ingress>

## Endepunkt i ulike miljøer

* Produksjonsmiljø: https://sparql.fellesdatakatalog.digdir.no
* Demomiljø:  https://sparql.demo.fellesdatakatalog.digdir.no

## Dokumentasjon

[Her kan du lese mer om SPARQL(W3C)](https://www.w3.org/TR/sparql11-overview/)
