# SPARQL API

<Ingress data-size='xs'>
    SPARQL-endepunktet er det kraftigaste søkeverktøyet på data.norge.no og gjev deg moglegheit til å søke på alle
    eigenskapar som finst i katalogane, så lenge du beherskar syntaksen.
</Ingress>

## Endepunkt i ulike miljø

- Produksjonsmiljø: https://data.norge.no/sparql
- Demomiljø: https://demo.fellesdatakatalog.digdir.no/sparql

## Dokumentasjon

[Her kan du lese meir om SPARQL(W3C)](https://www.w3.org/TR/sparql11-overview/)
