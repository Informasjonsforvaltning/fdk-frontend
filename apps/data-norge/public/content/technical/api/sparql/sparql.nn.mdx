# SPARQL API

<Ingress size='xs'>
    SPARQL-endepunktet er det kraftigaste søkeverktøyet på data.norge.no og gjev deg moglegheit til å søke på alle
    eigenskapar som finst i katalogane, så lenge du beherskar syntaksen.
</Ingress>

## Endepunkt i ulike miljø

- Produksjonsmiljø: https://sparql.fellesdatakatalog.digdir.no
- Demomiljø: https://sparql.demo.fellesdatakatalog.digdir.no
