# Catalog view API

<Ingress size='xs'>
    Lookup-endpoint for the organization's catalog i registrering service. Through this endpoint, organizations may
    extract all the information about the resources they have registered in the registrationr service. At the moment it
    is only possible to use this endpoint to extract information about the concept catalog.
</Ingress>

## Endpoint in different environments

- Production environment: https://catalog-view.api.fellesdatakatalog.digdir.no
- Demo environment: https://catalog-view.api.demo.fellesdatakatalog.digdir.no

## Documentation

[OpenAPI-specification for the catalog view endepoint (Github)](https://github.com/Informasjonsforvaltning/catalog-view-api/blob/main/openapi.yaml)

## How to gain access

The API is protected by Maskinporten. [contact us](/contact) if you want to gain access to extract the descriptions of your organization from the registration service.

Go to [the page about Maskinporten on Samarbeidsportalen (in Norwegian)](https://samarbeid.digdir.no/maskinporten/maskinporten/25) to read more about what Maskinporten is and how to get started as a consumer.
