# API

<Ingress size='xs'>Her finner du informasjon om endepunktene du kan bruke til å hente ut data.</Ingress>

Det er to ulike måter å søke etter data på Data.norge.no gjennom endepunkt. De er basert på hver sin teknologi. I tillegg tilbyr vi et oppslags-endepunkt hvor man kan hente ut egen virksomhets beskrivelser fra registreringsløsningen. 

## Søk etter data på Data.norge.no

### Søk

Søkemotoren på Data.norge.no er basert på Elastic search. Denne tjenesten søker i tekst i et begrenset antall felt og har filtre og annen avansert funksjonalitet som hjelper brukerne med å finne frem.

[Les mer om søkeendepunktet](/nb/technical/api/search).

### SPARQL

SPARQL-søket er basert på rdf-spørrespråket SPARQL, og er vårt mest avanserte og kraftfulle søk. Brukeren er nødt til å lage spørringer som følger korrekt syntaks, men vil kunne søke i alle høstede datapunkter, ikke kun forhåndsvalgte felt. Endepunket er åpent for oppslag, men man kan ikke skrive til det.

[Les mer om SPARQL-endepunktet](/nb/technical/api/sparql).

## Hent ut data fra registreringsløsningen. 

### Catalog View

Virksomheter som bruker registreringsløsningen, kan hente ut egne beskrivelser. Foreløpig er det kun mulig å hente ut begrep fra registreringsløsning. API-et er beskyttet med Maskinporten. 

[Les mer om Oppslags-endepunktet mot registreringsløsningen](/technical/api/catalog-view)
